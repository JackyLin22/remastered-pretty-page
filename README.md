# Prince George's County Food Safety Explorer

## Overview

The Prince George's County Food Safety Explorer is an interactive web application that allows users to search and visualize restaurant food safety data throughout Prince George's County, Maryland. Using publicly available inspection data, users can locate restaurants on an interactive map, search by restaurant name, and review food safety information.

This project was developed for INST630 (Programming Human-Centered Interfaces) and demonstrates data visualization, API integration, geographic mapping, and user-centered web design. I'm remastering this project I did back in September 2024 to create a more accessible page for everyone. Can look at my repository on [Making-a-Page-Pretty](https://github.com/JackyLin22/Making-a-Page-Pretty) as a starting point.

## Features

Interactive map powered by Leaflet.js

Search restaurants by name

Display restaurant locations throughout Prince George's County

View restaurant details including:
- Restaurant name
- Address
- Restaurant category
- Inspection information
  
Dynamic map markers and popups

Responsive design for desktop and mobile devices

Accessible interface with semantic HTML and descriptive labels
